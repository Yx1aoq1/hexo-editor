# hexo-editor

vue + ant-design-vue + express hexo博客后台管理

## 使用

```
https://github.com/Yx1aoq1/hexo-editor.git
cd hexo-editor
npm install
npm run build
npm run serve:app
```

  port: 3000, // 项目启动端口
